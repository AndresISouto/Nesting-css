# Nesting y text-wrap

## Uso del operador & con clases anidadas y justificación del texto

El nestig es una reciente actualización de css que permite la definición de estilos en clases anidadas de forma mas legible. No aporta ninguna funcionalidad nueva pero facilita la lectura y escritura de nuestros ficheros css al no tener que escribir reglas con largas concatenaciones y poder encadenarlas entre bloques {} para una mejor lectura.

El atributo @scope se usa de forma similar al nesting si solo tiene una raíz pero permite un mayor control al poder declarar el punto en el que deja de aplicarse.

El text-wrap ayuda a mantener una mejor presentacion tanto en titulos como en bloques de texto al ajustar el contenido.
